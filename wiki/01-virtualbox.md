# 1.7. Edit Network Interfaces

##### Execute:

```bash
sudo nano /etc/network/interfaces
```

```
auto lo
iface lo inet loopback

# Adapter 1: NAT
auto enp0s3
iface enp0s3 inet dhcp

# Adapter 2: Host-Only
auto enp0s8
iface enp0s8 inet static
address 192.168.56.10
netmask 255.255.255.0
```
